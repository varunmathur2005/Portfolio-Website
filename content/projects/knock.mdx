**Knock-Knock** is an AI-powered web application designed to simplify the job application process for students. It streamlines lead generation, cold emailing, and custom cover letter creation, significantly reducing manual effort.

## Features

- ### **Landing Page and Dashboard**  
  An intuitive and user-friendly interface with **Clerk** authentication, making access smooth and secure.

  ![Landing Page](https://github.com/user-attachments/assets/25733e4d-df53-4dbb-9f21-b116e67afe7c)
  
  ![Dashboard](https://github.com/user-attachments/assets/f69667d5-c2ab-4835-8d2b-bde444510f52)

- ### **Apollo Web Scraper**  
  A **Google Chrome plugin** that scrapes leads from Apollo.io and lets users easily download the data.

  [![Watch Demo](https://img.youtube.com/vi/pqTQKKBMGH4/0.jpg)](https://youtu.be/pqTQKKBMGH4)  
  _Click the thumbnail to watch the demo._

- ### **Cold Email Automation**  
  Automatically sends personalized, job-specific emails to each contact, saving time and effort in job outreach.

  [![Watch Demo](https://img.youtube.com/vi/lN0p4wB3y30/0.jpg)](https://youtu.be/lN0p4wB3y30)  
  _Click the thumbnail to watch the demo._

- ### **Cover Letter Generator**  
  Generates tailored cover letters that align with job descriptions, improving the chances of success.

  [![Watch Demo](https://img.youtube.com/vi/afmzuFTlE9M/0.jpg)](https://youtu.be/afmzuFTlE9M)

## Tech Stack

- **Frontend:** Next.js, React.js, ShadCN UI
- **Backend:** Node.js, Express.js
- **Authentication:** Clerk
- **Database:** MySQL, AWS S3
- **Web Scraping Plugin:** Google Chrome Plugin
- **Emailing:** SMTP
- **Cover Letter & Cold Email Generation:** OpenAI API
